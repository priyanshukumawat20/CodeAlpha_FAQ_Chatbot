# 🤖 CodeAlpha_FAQ_Chatbot

## Task 2 - Artificial Intelligence Internship at CodeAlpha

A simple FAQ Chatbot that answers questions related to the CodeAlpha AI Internship.

## 🔧 Tech Stack
- Python
- Scikit-learn (TF-IDF + Cosine Similarity)
- Gradio (for UI)

## ✨ Features
- Answers common questions about the CodeAlpha internship
- Uses NLP-based similarity matching to find the best answer
- Clean and interactive chatbot interface

## 🚀 How to Run
1. Install dependencies: `pip install scikit-learn nltk gradio`
2. Run the notebook in Google Colab or Jupyter
3. Use the Gradio link to access the chatbot
